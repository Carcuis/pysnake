# PySnake
![banner](resources/img/banner.png)

A python snake game based on pygame.

---

## Requirements

| Package       | version |
|---------------|---------|
| pygame        | latest  |
| opencv-python | latest  |
| torch         | 1.13.1  |

* Windows

  ```bat
  python main.py
  ```

* Linux & Darwin

  ```bash
  python3 main.py
  ```

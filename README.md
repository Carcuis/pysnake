# PySnake
![banner](resources/img/banner.png)

A python snake game based on pygame.

---

## Requirements

| Package       | version     |
| ------------- | ----------- |
| pygame        | >= 2.1.2    |
| opencv-python | >= 4.5.1.48 |

## Run

* Windows

  ```bat
  python main.py
  ```

* Linux & Darwin

  ```bash
  python3 main.py
  ```
